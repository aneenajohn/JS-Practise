# VAR Vs Let Vs Const

## Scoping:
  - Var has function scope 
  - Let and Const have block scope.

## Redeclaration:
 - Var can be redeclared 
 - let and const can be redeclared

## Redefinition:
 - Var and let can be redefined
 - Const cant be redefined
